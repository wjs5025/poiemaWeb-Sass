# 2. CSS를 프로그래밍 언어와 같이 작성할 수 있게 확장한 SassScript

# Sass Script

- CSS에서는 불가능한 연산, 변수, 함수 등의 확장 기능을 의미

# 데이터 타입 (예시)

- 숫자형 : 1.2, 13, 10px 등
- 문자열 : 'Lucida', "전인혁" 등
- 컬러 : blue, #04a3f9, rgba(255, 0, 0, 0.5) 등
- boolean : true, false
- null : null
- list : 1.5em 1em 0 2em, Helvetica, Arial, sans-serif
- map : (key1 : value1, key2 : value2)
